# Office Viewer Desktop 0.5.1 — Layout Hotfix

Release date: 2026-07-22

This patch release fixes the desktop workspace failing to fill the window after Windows DPI or viewport changes.

## Changes

- Make the desktop shell follow the dynamic viewport height.
- Pin the header, tab strip, document workspace and status bar to explicit grid rows.
- Prevent the fixed AI assistant from displacing the status bar.
- Keep the Markdown editor-theme picker within its compact height.
- Add Electron regression assertions for viewport and status-bar alignment.
- Make the Windows Release workflow upload versioned installers without hard-coded filenames.
