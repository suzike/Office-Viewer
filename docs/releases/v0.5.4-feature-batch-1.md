# Office Viewer Desktop 0.5.4 — Feature Batch 1

Release date: 2026-07-23

First feature batch from the three module roadmaps (`docs/plans/`): 15 new features across Markdown, HTML and the AI assistant.

## Markdown

- Print (context menu / Ctrl+Cmd+P) via a hidden sandboxed print window.
- Export long-image PNG: full-page document rendering written next to the source.
- TOC toolbar button: inserts/updates a `[toc]` marker (fence-aware, idempotent), rendered live by Vditor and expanded on export.
- Word count chip: CJK-aware characters/words/reading-time, debounced.

## HTML

- CodeMirror 6 source editor (highlighting, auto close tags/brackets, line numbers, search keymap) replacing the plain textarea.
- Format source (built-in formatter, `desktop/shared/html-format.ts`), preview zoom 50–200%, six insert snippets with tab stops.
- Export PDF through a new hidden-window IPC pipeline.

## AI assistant

- Regenerate last answer; export conversation as Markdown; per-message stats (first-token / total time, characters).
- Context extraction indicator (strategy, extracted/source characters, truncation badge).
- Slash commands in the composer with keyboard navigation.
- Five new quick actions: 改写语气, 压缩篇幅, 生成会议纪要, SWOT 分析, 对比选段与全文.
