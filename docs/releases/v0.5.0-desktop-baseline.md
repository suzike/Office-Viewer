# Office Viewer Desktop 0.5.0 — Baseline Release

这是 Office Viewer 作为独立 Windows 桌面应用维护的首个公开基线版本。

## 重点变化

- 独立 Electron 桌面宿主，不依赖 VS Code；
- 完整保留 Word、Excel、PowerPoint、Markdown、PDF、图片、电子书、压缩包和开发者格式能力；
- 多文档标签、Windows 文件关联、拖放、最近文件、单实例转交和原子保存；
- 内置 Git History 工作区；
- 新增文档 AI 助手，支持 Codex CLI、Claude Code、Ollama、DeepSeek、Kimi、OpenAI 兼容接口、Anthropic 与 Gemini；
- 新增 Office 解析缓存、Worker 后台解析、AI 上下文缓存和真实桌面性能基准；
- 增加安全文件会话、路径穿越防护、压缩炸弹限制、私网访问授权和密钥安全存储；
- 修复 Windows 位图剪贴板粘贴及 Markdown 模式切换误报未保存状态的问题。
