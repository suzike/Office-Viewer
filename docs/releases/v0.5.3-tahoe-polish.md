# Office Viewer Desktop 0.5.3 — Tahoe Polish and Interaction Fixes

Release date: 2026-07-23

This release fixes a comprehensive frontend-interaction audit (including two data-loss class bugs), advances the shell toward the macOS 26 "Tahoe" Liquid Glass language, and ships 90-item feature roadmaps for the Markdown, HTML and AI assistant modules.

## Bug fixes

- Rework the Office host bridge into an activation stack keyed by the visible document — hidden keep-alive viewers can no longer hijack saves or leave a document's save silently discarded (data-loss class).
- Compute dirty-close confirmation against fresh document state; tabs opened while the confirm dialog is up are no longer discarded.
- Restore mouse window dragging on the frameless window; only real file drags trigger the drop overlay.
- Document tabs, recent files and open-files now leave Git History; Git History no longer aborts assistant requests or leaks its selection into prompts.
- Close menus on outside click; show open/sync errors while a document is open (floating banner); re-register recent files via the new `openPaths` IPC so metadata and file watchers refresh.
- Assistant: throttled streaming Markdown rendering, scroll pinning that respects reading position, retry without duplicate bubbles or lost selection context, external link handling, provider-availability guard on quick actions, pointer-cancel resize cleanup.
- About dialog now shows the real version injected from package.json at build time.

## Design (macOS 26 Tahoe / Liquid Glass)

- Specular highlights (`--spec`) on all raised glass surfaces; deeper translucency (panel 78%, header 72%); radii scale 8/10/16px; asymmetric iMessage bubbles.
- Floating glass tab strip and metadata bar; spring motion curves; menu/message enter animations; 300 ms theme crossfade; layered welcome lighting.
- Windows 11 Mica `backgroundMaterial` (build 22621+, transparent-shell CSS fallback elsewhere).
- Typography nudged to macOS defaults; icons moved to 1.5px stroke with a rounded app mark.

## Roadmaps

- `docs/plans/feature-roadmap-markdown.md`, `feature-roadmap-html.md`, `feature-roadmap-assistant.md` — 30 prioritized features each with effort estimates and batch suggestions.
